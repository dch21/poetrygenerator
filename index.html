Instructions

Poetry Editor is a creative writing text editor. Writers can submit their writing onto the editor and manipulate the text using the filters on the left. The editor can create blackout/erasure poetry, change text appearance, use images as a backdrop, and features a built in word generator. Filters are included below to give the work a certain style.
